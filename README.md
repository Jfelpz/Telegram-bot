# Telegram-bot
bot que vai enviar posts para um canal de promoções no telegram

======================LOGICA DOS ARQUIVOS=======================================

bot.py → Junta todas as informações dos outros arquivos e envia para o bot
logic.py → decide o que enviar
telegram.py → envia mensagem
sheets.py → acessa Google Sheets
config.py → centraliza variávei
cache.py → controla quando um produto deve ser atualizado.
coletor.py → estrutura pronta para receber o Amapulse.
